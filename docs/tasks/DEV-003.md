# DEV-003

## Title

Create Project

---

## Epic

EPIC-1 Project Management

---

## Priority

High

---

## Estimate

2 hours

---

## Description

Реализовать создание проекта.

После нажатия кнопки "📝 Новый проект" пользователь вводит описание.

На основании описания создается новый проект.

---

## User Story

Как системный аналитик

Я хочу создать новый проект

Чтобы работать с требованиями отдельно от других проектов.

---

## Functional Requirements

### FR-1

Пользователь может нажать кнопку

📝 Новый проект

---

### FR-2

Бот ожидает описание проекта.

---

### FR-3

После получения текста создается новый Project.

---

### FR-4

Project получает статус Draft.

---

### FR-5

Проект отображается пользователю.
