# DEV-004

## Title

Project List

## Description

Добавить возможность просмотра списка проектов.

## User Story

Как системный аналитик,

я хочу видеть все свои проекты,

чтобы быстро открыть нужный.

## Functional Requirements

FR-1

Добавить кнопку:

📂 Мои проекты

FR-2

Если проектов нет:

У вас пока нет проектов.

FR-3

Если проекты есть:

Показать список:

1. Название
Статус

2. Название
Статус

...

## Acceptance Criteria

Нажатие "📂 Мои проекты" отображает все сохраненные проекты.

Используется ProjectService.

Handler не содержит бизнес-логики.
